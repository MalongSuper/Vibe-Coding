How to Play

Game Rules

• Select a game mode: Addition or Subtraction

• For Addition, choose the number of terms (2, 3, or 4)

• Solve the math problems as quickly as you can

• Enter your answer and click Submit

• Get it right to earn points and move to the next question

• Get it wrong and the game ends

• Empty answers won't end the game - try again!

Power Ups

• 🧮 Calculator: Get the right answer immediately (+1 point)

• 🔢 Digits: Remove one random digit from a number to make it easier

• 🚀 Boost: Next correct answer gives you 5 points instead of 1

• Each power can only be used once per game

Timer

• You are not timed because the timer goes up
